eLife API Problem PHP
=====================

This library provides a [RFC 7807 handler](https://tools.ietf.org/html/rfc7807) for the eLife Sciences applications.

Dependencies
------------

* [Composer](https://getcomposer.org/)
* PHP 7

Installation
-------------

`composer require elife/api-problem`

Running the tests
-----------------

`vendor/bin/phpunit`
